# TERM-CNODE
Is A Simple Chat App Using NodeJS + Socket.io.

# How To Use
For To Use Is App Is Very Simple You Must Run A Simple Syntax In Terminal

### For Server NodeJS
```
node server
```

### For Client Chat App
``` 
node client {username}
```
